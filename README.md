# 🎓 CourseHub - Online Course Management System

A full-stack "E-Learning Course Management Platform" built using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.  
CourseHub provides a complete solution for managing online courses with separate "Admin" and "Student dashboards", course enrollment, and progress tracking features.

##  Overview

CourseHub is designed to simplify online learning management by allowing students to explore courses, enroll, and monitor their learning progress while administrators can efficiently manage students, courses, and enrollments.

This project follows a client-server architecture with REST APIs and database integration.
